# PUZ Reader

A .PUZ file decoder and encoder based on the format specified [here](https://code.google.com/archive/p/puz/wikis/FileFormat.wiki).

## Features

- Encode and Decode .PUZ Files
- Unscramble and Re-scramble .PUZ files
- Preserve all data
